# @example/fss-shell-example-lib

One line on what this library does and who it's for. (Replace this, and the
package name above, when you scaffold a real library.)

## Install

```bash
pnpm add @example/fss-shell-example-lib
```

## Quick start

```ts
import { exampleGreeting } from "@example/fss-shell-example-lib";

exampleGreeting("world");
// → "Hello from the example library, world."
```
