# mp-lb-run (tombstone)

(`zap arc deploy`, see the zapper repo's `docs/tech/arc.md`) — on **ArcNet**,
MAP Lab's network (config in `mgr/arcnet/`; how-to in `mgr/processes/deploy.md`).

proven Terraform from this store lives on as zapper's bundled arc modules.

is in the stackforge git repo if ever needed.
